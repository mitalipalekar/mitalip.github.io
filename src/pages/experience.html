NASA Ames, Human Systems Integration Lab

Software Development Intern

I worked with the Internet of Things using ESP chips, MQTT communication protocols and Augmented Reality through the Microsoft HoloLens to create Procedure Execution Software for the International Space Station. I implemented spatial mapping for path finding, UI menu selection for procedure selection and placing of animated holograms for in-procedure execution using the Microsoft Hololens.

UW CSE Security & Privacy Lab

Undergraduate Research Assistant

I am working on developing and maintaining Confidante, an encrypted email client that makes sending PGP encrypted emails easy by using Keybase for automatic key management. I have worked on implementing several features such as making private key signing optional, modifying the front-end to make interactions with past email threads possible while composing new messages as well as adding the drafts functionality to the email client.

Georgian Digital Text Collective

Web Developer

As a both the front and back end Web Developer, I have developed PHP scripts to convert Georgian to XML and JavaScript scripts to transpose XML to XHTML. Additionally, I have also proposed, designed and implemented UI design changes to the front-end of our website and I also manage the social media for our group.
